Vortex-ARE is a state-of-the-art **Autonomous Revenue Engine** designed to operate as a headless, parasitic intelligence layer on top of modern LLMs. It coordinates **Gemini (Thinking)** and **ChatGPT** to autonomously ideate, architect, and implement digital products with a primary financial target of **10,000 NTD**.

## 🏗️ Architecture: Synesthesia-Fusion

Vortex-ARE uses the **Synesthesia-Fusion** design pattern, where disparate system signals (telemetry, AI responses, resource usage) are unified into a single "Resonance Score."

- `headless_bridge.py`: The data-conduit between the OS and the web-based LLMs.
- `system_engine.py`: The central nervous system, handling multi-threaded "NeuralHub" event propagation.
- `revenue_engine.py`: The monetization layer, tracking the 10,000 NTD goal.
- `dashboard_gui.py`: The visual interface for real-time monitoring.

## 🚀 Quick Start

### 1. Prerequisites
- Python 3.10+
- Chrome Browser with logged-in sessions for Gemini & ChatGPT.
- Playwright: `pip install playwright && playwright install`

### 2. Installation
```powershell
# Clone the repository
git clone https://github.com/zephyus/Vortex-ARE.git
cd Vortex-ARE

# Install dependencies
pip install -r requirements.txt
```

### 3. Execution
```powershell
python main_app.py
```
